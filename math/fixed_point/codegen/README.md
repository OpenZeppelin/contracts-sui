# `openzeppelin_fp_math` Codegen

Offline tooling that produces the AAA-rational coefficient tables and
deterministic test vectors consumed by `math/fixed_point/sources/internal/`
and the per-type test directories.

The Move package itself is fully self-contained at runtime; this directory is
**only** invoked when coefficients need to be (re)derived.

## Requirements

- Python ≥ 3.10
- `mpmath ≥ 1.3`, `scipy ≥ 1.13`, `numpy ≥ 1.26`

## Install

From the repo root:

```sh
python3 -m venv codegen/.venv
source codegen/.venv/bin/activate
pip install -e codegen
```

## Generate (CDF)

```sh
python -m codegen.cdf.derive            # AAA fit + degree sweep → .derive_output.json
python -m codegen.cdf.emit_coefficients # → math/fixed_point/sources/internal/cdf_coefficients.move
python -m codegen.cdf.emit_test_vectors # → tests/{sd29x9_tests,ud30x9_tests}/cdf_test_vectors.move
python -m codegen.cdf.validate          # re-checks the committed coefficient module against scipy
```

`derive.py` writes a JSON intermediate (`codegen/cdf/.derive_output.json`,
gitignored) that the `emit_*.py` scripts consume. `validate.py` does **not**
re-derive — it parses the committed Move file and re-runs the on-chain integer
arithmetic in Python to assert the quantized error bound.

CI runs only `validate.py`. Regeneration is local-only.

## Layout

```
codegen/
├── shared/
│   ├── reference.py     # mpmath oracle for Φ at 100 dps
│   └── move_emit.py     # Move literal / banner helpers
└── cdf/
    ├── derive.py
    ├── emit_coefficients.py
    ├── emit_test_vectors.py
    └── validate.py
```

Future `pdf/` and `inverse_cdf/` directories follow the same shape.

## Generated Files

Both committed Move files carry an `AUTO-GENERATED` banner and must not be
hand-edited; regenerate via the steps above.

| Path | Generated by |
|------|--------------|
| `math/fixed_point/sources/internal/cdf_coefficients.move` | `codegen/cdf/derive.py` + `codegen/cdf/emit_coefficients.py` |
| `math/fixed_point/tests/sd29x9_tests/cdf_test_vectors.move` | `codegen/cdf/emit_test_vectors.py` |
| `math/fixed_point/tests/ud30x9_tests/cdf_test_vectors.move` | `codegen/cdf/emit_test_vectors.py` |
